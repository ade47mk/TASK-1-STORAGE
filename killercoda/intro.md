# Welcome to the Kubernetes Persistent Storage Exam! 🎓

In this hands-on exam, you will learn how to work with persistent storage in Kubernetes.

## 📚 What You'll Learn

- Creating PersistentVolumeClaims (PVC)
- Configuring storage classes and access modes
- Mounting persistent volumes in Pods
- Verifying volume creation and persistence

## 🎯 Exam Task

**A developer needs a persistent volume for an application.**

Your task is to:
1. Create a PersistentVolumeClaim with specific requirements
2. Create a Pod that uses this PVC
3. Verify that data persists across Pod restarts

## 📋 Requirements

- **Size**: 100Mi
- **Access Mode**: ReadWriteOnce
- **Storage Class**: local-path
- **Mount Path**: /data
